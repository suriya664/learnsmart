# Homepage Font Sizes & Typography Guide

## Current Font Size Analysis

### Typography System

**Font Families:**
- **Headings**: `Poppins` (Google Fonts) - Bold, modern
- **Body Text**: `Inter` - Clean, readable
- **Brand**: `Sans-serif` fallback

---

## Heading Sizes (All using `Poppins`)

### H1 - Main Hero Title
- **Class**: `display-4`
- **Size**: `2.5rem` (40px)
- **Weight**: `700` (Bold)
- **Usage**: Hero section main headline
- **Line Height**: `1.3`

### H2 - Section Titles
- **Class**: Standard `h2` or custom
- **Size**: `2rem` (32px)
- **Weight**: `600` (Semi-bold)
- **Usage**: "Choose Your Learning Path", "Most Popular Courses", etc.
- **Line Height**: `1.3`
- **Example**: "Most Popular Courses", "Why LearnSmart Academy?"

### H3 - Subsection Titles
- **Size**: `1.75rem` (28px)
- **Weight**: `600`
- **Usage**: Stats numbers, course titles
- **Line Height**: `1.3`

### H4 - Small Headings
- **Size**: `1.5rem` (24px)
- **Weight**: `600`
- **Usage**: Card titles, feature names
- **Line Height**: `1.3`
- **Example**: "Web Development", "Lifetime Access"

### H5 - Card Titles
- **Size**: `1.25rem` (20px)
- **Weight**: `600`
- **Usage**: Course card titles, category names
- **Example**: "Complete Web Development Bootcamp"

### H6 - Smallest Headings
- **Size**: `1rem` (16px)
- **Weight**: `600`
- **Usage**: Minor headings

---

## Body Text Sizes

### Large Body Text (Hero)
- **Class**: `lead`
- **Size**: `1.25rem` (20px)
- **Weight**: `400`
- **Line Height**: `1.5`
- **Usage**: Hero description paragraphs

### Standard Body Text
- **Class**: `p` or no class
- **Size**: `1rem` (16px) - Default
- **Weight**: `400` (Regular)
- **Line Height**: `1.6`
- **Usage**: General paragraphs, descriptions

### Small Text
- **Class**: `small` or `text-muted`
- **Size**: `0.875rem` (14px)
- **Usage**: Metadata, timestamps, secondary info

### Extra Small Text
- **Class**: `small` + custom
- **Size**: `0.75rem` (12px)
- **Usage**: Labels, badges text

---

## Button Text Sizes

### Large Buttons (`btn-lg`)
- **Size**: `1.125rem` (18px)
- **Weight**: `500` (Medium)
- **Padding**: `0.75rem 1.5rem`
- **Usage**: Hero CTAs, major actions

### Standard Buttons (`btn`)
- **Size**: `1rem` (16px)
- **Weight**: `500`
- **Padding**: `0.75rem 1.5rem`
- **Usage**: Standard actions

### Small Buttons (`btn-sm`)
- **Size**: `0.875rem` (14px)
- **Weight**: `500`
- **Padding**: `0.5rem 1rem`
- **Usage**: Secondary actions, card buttons

## Badge & Label Sizes

### Badges

- **Weight**: `500`
- **Padding**: `0.5rem 1rem`

## Typography Hierarchy Summary

```
1. Hero H1 (40px) - Primary attention
2. Section H2 (32px) - Major sections
3. Card H5 (20px) - Course/category names
4. Body Lead (20px) - Important paragraphs
5. Body Standard (16px) - Regular text
6. Small (14px) - Supporting info
```

## Color & Contrast

**Text Colors:**
- Primary: `#0066cc` (Blue) - Links, CTAs
- Dark: `#2c3e50` (Dark Gray) - Headings
- Muted: `#6c757d` (Gray) - Secondary text
- Light: `#ffffff` (White) - On dark backgrounds

**Background Colors:**
- White: `#ffffff` - Main background
- Light Gray: `#f8f9fa` - Section backgrounds
- Primary: `#0066cc` - Hero, CTAs

## Spacing

**Margins:**
- H2: `margin-bottom: 1.5rem` (24px)
- H5: `margin-bottom: 0.75rem` (12px)
- Paragraphs: `margin-bottom: 1rem` (16px)

**Line Heights:**
- Headings: `1.3` - Tight, impactful
- Body: `1.6` - Comfortable reading
- Leads: `1.5` - Balanced
